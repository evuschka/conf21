# Dependency Visualizer - Этап 1

Инструмент визуализации графа зависимостей для менеджера пакетов.

## Описание

Минимальный прототип CLI-приложения для визуализации зависимостей npm-пакетов. 
На данном этапе реализована только конфигурация приложения через командную строку.
